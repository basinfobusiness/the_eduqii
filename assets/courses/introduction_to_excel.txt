**Functions**

Functions in Excel are built-in (ready-made) formulas used to perform calculations. 

These functions make it easier to perform calculations and analyses data in Excel. For 

example, if you are to add values in cells ranging from A1 to A20, a function can be 

used to reduce the stress of repeating individual cells in the formula. So instead of 

=A1+A2+A3+A4+A5……A20, you can use =SUM (A1:A20) to find the total of 

values in cell A1 to A20
